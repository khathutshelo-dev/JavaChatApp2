# Java Chat Application

A real-time desktop chat application built using Java sockets, multithreading, and Swing GUI.

## Features
- Real-time messaging
- Multi-client support
- Online users panel
- Modern dark UI
- Message bubbles
- Emoji support
- Multithreading
- Socket networking

## Technologies Used
- Java
- Java Swing
- Socket Programming
- Multithreading

## How To Run

### Compile
```bash
javac src/chatapp/*.java
